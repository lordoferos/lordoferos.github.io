A user-function for creating Word reports in R

R can be used to create custom reports complete with tables and graphs. The packages needed to achieve this include `tidyverse` and `officer`.

Getting Started

To get started with creating reports in R, you'll need to install the appropriate packages. You can do this by running the following code in the R console:

install.packages(c("tidyverse", "officer"))

Once you have the packages installed, you can load them into your R session using the following code:

library(ggplot2)
library(tidyverse)
library(officer)

Creating a Simple Report

Let's create a simple report using the iris dataset that is available in R:

library(tidyverse)
iris_df <- iris

Next, we'll create a basic summary the data:

#create summary table, grouped by a specific variable
rep = describeBy(iris_df, group=iris_df$Species)

This will create a summary based on the species variable.

Conclusion

R is a powerful tool for creating reports. It's possible to create beautiful, informative reports using these packages.
